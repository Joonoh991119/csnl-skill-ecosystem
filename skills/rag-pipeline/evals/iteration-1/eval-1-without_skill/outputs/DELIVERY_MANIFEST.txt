================================================================================
ARCHITECTURE
================================================================================

Data Flow:
  Zotero API → 02_ingest_papers.py → PostgreSQL
       ↓
  Local documents → 03_chunking_and_embeddings.py → OpenAI API
       ↓
  Embeddings → PostgreSQL (pgvector index)
       ↓
  User Query → 04_rag_query_engine.py → Semantic search
       ↓
  Top K passages → GPT-4 Turbo → Synthesis
       ↓
  Response with citations

Database Schema:
  papers (50 rows):
    - id, zotero_id, title, authors, year, abstract, doi, url
  
  chunks (5000-10000 rows):
    - id, paper_id, chunk_text, chunk_index, embedding[1536]
  
  search_history (tracking):
    - id, query, num_results, execution_time_ms, timestamp

Indexes:
  - IVFFlat index on chunks.embedding (vector_cosine_ops)
  - Lists=100 (optimized for your dataset size)
