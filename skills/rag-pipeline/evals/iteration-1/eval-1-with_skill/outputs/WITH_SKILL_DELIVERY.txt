================================================================================
USAGE WORKFLOW
================================================================================

Step 1: Setup (2 minutes)
  python setup_pipeline.py
  - Creates PostgreSQL database
  - Enables pgvector extension
  - Creates 5 tables with EMBEDDING_DIM=1024
  - Generates rag_config.json

Step 2: Ingest Papers (10 minutes)
  python ingest_from_zotero.py --sample-papers 50
  - Loads embedding model on MPS device
  - Creates section-aware chunks from 50 sample papers
  - Generates embeddings for ~8,000 chunks
  - Stores in PostgreSQL with metadata

Step 3: Query (Interactive)
  python hybrid_search.py
  - Loads embedding model on MPS
  - Runs dense + sparse searches
  - Combines results via RRF
  - Returns ranked findings
